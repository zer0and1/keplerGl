# Node/Express

This example shows how to embed Kepler.gl in a node/express/webpack application.

## 1. Install

```bash
npm install
```

or

```bash
yarn --ignore-engines
```

## 2. Mapbox Token

add mapbox access token to node env

```bash
export MapboxAccessToken=<your_mapbox_token>
```

## 3. Start the app

```bash
npm start
```
